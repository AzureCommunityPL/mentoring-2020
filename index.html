Azure Community Mentoring Program 2020

1 czerwca - 31 sierpnia 2020
Organizowany przez Microsoft Azure User Group Poland

Azure Community Mentoring Program 2020

To program mentoringowy przygotowany z myślą o wszystkich tych, którzy chcieliby rozwijać swoje umiejętności w obszarze rozwiązań platformy Microsoft Azure pod okiem doświadczonych mentorów. Program organizowany jest przez członków społeczności zebranej w ramach Microsoft Azure User Group Poland.

Program jest darmowy, niemniej wymagamy od Ciebie zaangażowania!

Dlaczego postanowiliśmy uruchomić taki program

TO DO

Dla kogo?

Program ten dedykowany jest do osób, które posiadają już doświadczenie w IT, ale chciałyby rozwinąć swoje umiejętności na platformie Microsoft Azure.

Jeśli zatem:

  • pracujesz na co dzień z Azure, ale chciałbyś rozwinąć się w konkretnym kierunku
  • lub masz doświadczenie z innymi dostawcami usług chmurowych i chciałbyś zacząć przygodę z Azure
  • czy też zajmujesz się zupełnie inną gałęzią IT, ale czujesz, że "chmura to jest to co chciałbyś robić

To nie czekaj długo i zapisz się do programu‼️

Czego możesz się spodziewać w ramach programu

Poza porządną dawką wiedzy, którą pomoże zdobyć Ci wybrany przez Ciebie Mentor liczymy, że będziesz się dobrze bawić. Dodatkowo, Organizatorzy planują kilka niespodzianek...no ale o nich nie możemy teraz napisać, bo nie byłyby niespodziankami.

Możesz aplikować od 11.05.2020 do 24.05.2020

25 - 31 maja

Wybór uczestników przez Organizatorów i Mentorów

Organizatorzy, po otrzymaniu zgłoszeń od potencjalnych uczestników programu, dokonują wstępnej selekcji. Następnie, wybrane, zanonimizowane zgłoszenia przekazywane są Mentorom wskazanych w zgłoszeniu. Mentorzy, na podstawie zanonimizowanych formularzy, wybierają dobrze rokujące zgłoszenia. Organizatorzy przekazują informację, o dostaniu się lub odrzuceniu zgłoszenia do programu. Mentorzy otrzymują kontakt do wybranych przez siebie osób.

Czas trwania programu
